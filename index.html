🌐 Opción para verlo desde otro dispositivo

Para que el cliente vea el código desde su propio celular, se necesita guardar el dato en internet. La forma gratis más práctica es usar un archivo JSON en GitHub.

El token no queda escrito en el HTML. Solo lo escribís vos en tu navegador. No compartás el link de admin con el token guardado.

Si el repositorio es público, el archivo JSON también será visible. No publiques información sensible, solo el código temporal y datos mínimos.
